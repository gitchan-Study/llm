# LLM 바닥부터 만들기

1. pre training으로 일반적인 언어 능력 학습 [잘 쓰여진 책] (인터넷에 있는 자료)
2. fine tuning 단계에서 특정 업무에 적응 (예. 질문-답변 data set)

4. 내부적으로 질의 반복해 더 좋은 결론 도출 (생각을 거듭에 거듭)
